Card

The Card component is a flexible and customizable component designed to display content in a "card-like" format.

  • It supports customizable headers, body content, and footers
  • as well as adjustable sizes and variants.
  • You can also make the card interactive by passing an onClick handler.

Example​

import { Button } from "ghewar";
import { Heading } from "ghewar";
import { Card } from "ghewar";

const App = () => {
  return (
    <div>
      <Card
        header={<Heading level={2}>Card Header</Heading>}
        body={<p>This is the body content of the card.</p>}
        footer={
          <Button size="lg" variant="secondary">
            Card Footer
          </Button>
        }
        size="medium"
        variant="primary"
        onClick={() => alert("Card clicked!")}
      />
    </div>
  );
};

export default App;

Card Variants​

The Card component supports two variants: primary and secondary.

  • If it's not specified then by default, the card uses the primary variant.

Card Action​

The Card component includes an action feature that lets you trigger a custom function when the card is clicked.

  • To enable the action feature, simply provide an onClick prop.
  • If the onClick handler is passed, a hover effect is applied to the card, indicating that it is interactive.

  <Card 
      header={<h3>Learn More</h3>} 
      body={<p>Click the card to learn more about this topic.</p>} 
      size = "medium"
      variant = "primary"
      onClick={() => window.location.href = 'https://www.google.com'} 
  />

Props​

PropTypeDefaultDescription
headerReactNodeundefinedOptional. Content for the header section of the card.
bodyReactNodeundefinedOptional. Content for the main body of the card.
footerReactNodeundefinedOptional. Content for the footer section of the card.
size"default" | "small" | "medium" | "large""default"Controls the size of the card. Choose from: small, medium, large.
variant"default" | "primary" | "secondary""default"Controls the appearance of the card. Choose from: default, primary, secondary.
onClick() => voidundefinedOptional. A function triggered when the card is clicked. Adds hover effect if provided.
